## Installation

Use the package manager [npm](https://www.npmjs.com/) to install foobar.

```bash
npm install foobar
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
